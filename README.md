# Snake-Cpp
A simple snake game using cpp for g++ and apple clang compiler using ncurses.

## Procedure
Enter the following code on a g++ compiler terminal window:<br/>
```
g++ -std=c++11 same.cpp -o run -lncurses
```

To run the file:<br/>
```
./run
```
